# Experience Log

## 约定

- 每次一个文件，文件名格式：`YYYY-MM-DD-主题.md`
- 内容尽量固定成四段：
  - 背景
  - 结论
  - 有效做法
  - 下次优化
- 只记录能复用的结论，不写流水账
- 如果同一个项目有多轮迭代，就把最终结论合并到同一篇里
